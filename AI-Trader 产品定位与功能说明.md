# AI-Trader 智能量化交易平台

## 产品定位

**AI-Trader** 是一款面向个人投资者的**AI 辅助量化交易 Web 平台**，以"策略驱动、AI 赋能、风控先行"为核心理念，集行情监控、策略回测、信号追踪、模拟/实盘交易与风险管理于一体，帮助用户在加密货币市场中进行科学、系统化的交易决策。

| 维度 | 定位说明 |
|------|---------|
| **目标用户** | 个人加密货币交易者、量化投资爱好者、AI 交易研究者 |
| **核心价值** | 降低量化交易门槛，通过 AI 信号+策略模板+风险控制三位一体，让非专业用户也能执行系统化交易 |
| **使用场景** | 日常盯盘与交易执行、策略研究与回测验证、风险监控与仓位管理、AI 辅助决策分析 |
| **差异化** | 终端极简 UI 风格、内置多策略 AI 信号体系、模拟/实盘一键切换、交易-策略-风控闭环 |

## 功能模块详解

### 一、交易模块 (Trading Hub)

**功能定位**：核心交易执行终端，支持市价/限价下单、持仓管理、挂单追踪

| 功能 | 说明 |
|------|------|
| **下单面板** | 选择交易对（BTC/ETH/SOL 等）、买卖方向、订单类型（市价/限价）、数量、杠杆倍数（1x-20x） |
| **杠杆风控联动** | 选择的杠杆超过风控设置的最大杠杆时，自动显示红色警告提示 |
| **持仓列表** | 实时展示当前持仓：标的、方向、数量、开仓价、标记价、盈亏、收益率 |
| **挂单管理** | 查看未成交/已成交订单，含订单类型、价格、数量、状态 |
| **信号推荐** | 右侧内置信号推荐卡片，展示当前最高置信度的交易信号，一屏完成决策-下单 |

---

### 二、信号广场 (Signal Square)

**功能定位**：多策略 AI 交易信号的聚合与筛选中心

| 功能 | 说明 |
|------|------|
| **多策略信号源** | 内置 4 大策略体系：动量突破、均值回归、趋势跟踪、网格交易，共 12 组实时信号 |
| **策略筛选** | 按策略类型筛选信号（全部/动量突破/均值回归/趋势跟踪/网格交易） |
| **策略分组展示** | 全部视图下按策略自动分组，每组带颜色标识（绿/蓝/橙/紫） |
| **信号卡片** | 每个信号以独立卡片展示：标的名称、多空方向、策略标签、历史胜率、入场价/目标价/止损价 |
| **关注系统** | 可关注/取消关注信号，关注列表置顶展示，支持一键清空关注 |
| **今日推荐** | 信号面板顶部高置信度信号推荐（如 BTC 动量突破 87分） |

---

### 三、策略中心 (Strategy Center)

**功能定位**：策略全生命周期管理 + 模拟交易结果看板

#### 3.1 策略管理

| 功能 | 说明 |
|------|------|
| **我的策略** | 已有策略列表：名称、类型、运行状态（运行中/已暂停）、收益率、夏普比率 |
| **新建策略** | 通过弹窗表单创建：策略名称、类型（5 种可选）、风险等级（低/中/高）、初始资金分配、交易标的选取 |
| **策略模板** | 内置 4 套模板（动量突破/均值回归/网格交易/统计套利），一键填入新建表单 |
| **启停控制** | 对已有策略执行启动/暂停操作，支持删除废弃策略 |

#### 3.2 模拟盘交易看板

| 功能 | 说明 |
|------|------|
| **业绩总览** | 4 大核心指标卡片：总收益率（+8.42%）、当前权益（$10,842）、胜率（62.3%）、夏普比率（1.35） |
| **权益曲线** | 内嵌 SVG 走势图，展示模拟账户净值变化轨迹（$10,000 → $10,842，10 日数据） |
| **最近模拟成交** | 8 笔已平仓模拟交易明细：标的、方向、入场/出场价、盈亏金额、收益率、成交时间 |
| **参照对比** | 模拟盘结果作为策略实际运行效果的参考基准，与实盘表现形成对比 |

---

### 四、市场情报 (Market Intelligence)

**功能定位**：多维市场数据聚合与分析

| 功能 | 说明 |
|------|------|
| **行情总览** | 多币种实时报价表：价格、24h 涨跌幅、成交量 |
| **标的选择** | 点击任意标的展开技术分析详情面板 |
| **技术指标** | 每个标的提供：RSI(14)、MACD、MA20/MA50 均线、布林带（上/中/下轨）、24h 最高/最低价、市值 |
| **图表可视化** | 内嵌 SVG 图表：RSI 仪表盘、MACD 柱状图、价格走势折线图、布林带带状图 |
| **操作建议** | 基于技术指标生成综合建议：买入/持有/观望 |
| **搜索过滤** | 按标的名称搜索过滤行情列表 |

#### 4.1 辅助信息面板（右侧 QuickInfoPanel）

| 功能 | 说明 |
|------|------|
| **市场情绪指数** | 综合评分（0-100）+ 情绪等级（恐惧/中性/贪婪）+ 趋势方向 |
| **当前预警** | 实时风险预警摘要，按严重程度标记（危险/警告/信息） |
| **AI 一句话建议** | AI 基于当前市场状态生成简明的交易建议 |

#### 4.2 新闻与事件

| 功能 | 说明 |
|------|------|
| **市场新闻** | 按重要性（高/中/低）排列的加密货币相关新闻，含时间戳 |
| **经济日历** | 未来重要事件清单：CPI 数据发布、议息会议、期货交割等，标注影响等级 |
| **宏观指标** | 美元指数、美债收益率、标普500、黄金价格等联动参考 |
| **涨跌榜** | 24h 涨幅/跌幅最大标的排行 |

### 五、交易组合 (Trading Portfolio)

**功能定位**：资产与风险全景仪表盘，模拟/实盘双模式

#### 5.1 模式切换

| 功能 | 说明 |
|------|------|
| **模拟/实盘双模式** | 一键切换模拟交易和实盘交易视图，资产数据、持仓、风险指标、预警信息同步切换 |
| **数据隔离** | 两种模式数据完全独立，互不干扰 |
| **顶部同步** | 切换模式时顶部栏资产数值、模拟/实盘标签颜色同步变更 |

#### 5.2 资产概览

| 指标 | 说明 |
|------|------|
| 总资产 | 当前账户总价值 |
| 可用余额 | 可自由支配资金 |
| 未实现盈亏 | 当前持仓浮动盈亏金额及百分比 |
| 胜率 | 历史交易胜率 |
| 总交易次数 | 累计交易笔数 |
| 利润因子 | 总盈利/总亏损比率 |

#### 5.3 持仓明细

- 完整持仓列表：标的、方向、数量、开仓价、标记价、未实现盈亏、收益率、占用保证金、杠杆

#### 5.4 风险指标

| 指标 | 说明 |
|------|------|
| 总敞口 | 当前总风险暴露金额及占总资产比例 |
| 单笔风险 | 每笔交易承担的风险百分比（建议 ≤ 2%） |
| 当前回撤 | 账户最大回撤比例，超过 5% 触发预警 |
| VaR(95%) | 日风险价值估算，衡量潜在最大损失 |

#### 5.5 风险预警

- 三级预警体系：🔴 危险（需立即处理）/ 🟡 警告（需要关注）/ 🔵 信息（常规提示）
- 示例预警：仓位集中度超标、止损位接近当前价、已实现盈亏报告

#### 5.6 止损止盈规则

| 规则 | 说明 |
|------|------|
| 默认止损 | 亏损达到设定比例时自动平仓 |
| 默认止盈 | 盈利达到设定比例时自动平仓 |
| 最大杠杆 | 允许使用的最大杠杆倍数（与交易模块联动） |
| 集中度限制 | 单个资产占总资产的最大比例 |

- 编辑面板：统一弹窗编辑所有风控参数，实时生效

## 交互设计特色

### 视觉风格
- **终端极简美学**：2px 实线边框、无圆角、无阴影、等宽字体（monospace）
- **高对比度配色**：黑底绿字（成功/做多）、红色（风险/做空）、黄蓝辅助
- **克制的信息密度**：数据优先，装饰元素极简，聚焦交易决策

### 交互亮点
- **三栏布局**：左侧导航 + 中间主内容 + 右侧信息面板，大屏效率最大化
- **模拟/实盘一键切换**：全平台数据随模式切换，体验无缝
- **信号关注机制**：关注信号置顶展示，不遗漏高价值交易机会
- **策略模板一键套用**：从模板到策略创建仅需两步
- **杠杆风控联动**：下单时杠杆超限即时警告，防风险于未然
- **键盘友好**：大量操作通过简洁按钮完成，适合高频操作场景
